# Facebook Ads Rules

## Safety
- NEVER modify live campaigns without explicit user confirmation
- NEVER increase budgets without confirmation
- NEVER delete anything - always pause instead
- Always double-check ad account selection before any action

## Campaign Structure
- Campaigns are organized by vertical (home services, mass torts, etc.)
- Ad sets are organized by targeting/audience
- Always ask which ad account if not specified
- Always ask which campaign/ad set if ambiguous

## Performance Benchmarks
> Update these with actual benchmarks as we learn them
- Flag ads with CTR below 1% after 1000 impressions
- Flag ads with CPL above 2x target after $50 spend
- Recommend scaling ads with CPL below target and 50+ leads

## Creative Rotation
- Pull creatives from Google Drive
- Landing page links come from Trello cards
- Test multiple creatives per ad set
- Kill underperformers fast, scale winners
