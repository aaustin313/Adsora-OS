1. Create your form from scratch.
2. Choose an Email Form field, which will automatically generate a submit button.
3. Click on the button to bring up the settings menu on the right side.
4. In the Text field of the button settings, type "GO TO STEP #2".
5. In the Subtext field, type any text you'd like but set the text to be transparent (e.g., using the color picker to set the opacity to 0%) to prevent it from overlapping with the text embedded in the CSS code.
6. Go to Options ➜ Advanced Settings ➜ Custom CSS.
7. Enter the custom CSS code provided below in the Custom CSS section.
8. If you want to change colors, fonts, or the subtext, I have marked those areas in red and just change those out to whatever you prefer


Should look like this:
